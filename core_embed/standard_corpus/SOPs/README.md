# Universal Standard Operating Procedures (SOPs)

This directory contains the templates for operating on a project governed by the Universal Governance Standard.

## Philosophy
We prefer boring, explicit, and auditable engineering. We do not use clever abstractions when a simple, readable function suffices.
The repository must remain clean, and AI agent changes must always be planned, validated, and logged.
